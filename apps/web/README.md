# School Management - React Client

This is the Single Page Application (SPA) frontend built with React, Vite, and TypeScript.

## Key Features & Stack
- **Styling**: Tailwind CSS & custom glassmorphism design variables.
- **Component Primitives**: Radix UI (ShadCN).
- **Interactive Drag-and-Drop Editor**: DnD Kit for absolute grid manipulation.
- **Data Hydration**: TanStack Query (React Query) for state fetching, caching, and background synchronizations.
- **Form State**: React Hook Form with Zod integration.
- **Transitions**: Framer Motion.

## Commands
- Run dev server: `npm run dev`
- Build production bundle: `npm run build`
- Preview production build: `npm run preview`
